# Phase 0

## Goal

Establish the repository scaffold and engineering discipline without implementing business logic.

## Deliverables

- repository directory structure aligned with the target architecture
- pinned Python project and developer tooling
- minimal FastAPI orchestrator service
- `/healthz` and `/readyz`
- `docker-compose.dev.yml`
- `.env.example`
- root `Makefile`

## Explicitly excluded

- database schema and migrations
- `research_task` APIs
- search, fetch, parse, index, verify, and reporting logic
